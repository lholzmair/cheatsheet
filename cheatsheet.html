This is a cheatsheet in html!

World Wide Web General


1. URL, DNS and HTML allowed any user worldwide to connect and habing access to any kind of data
2. URL - Universal Resource Locator
3. The components are:
  * protocol (<http> or <https> for example)
  * path
  * query string
  * fragment id

Webserver


1. A software with underlying hardware accepting HTTPs and delivering pages/data
2. It receives requests, processes them, retrieves the data and sends it to a user
3. It can save data in all kinds of files, data bases or CMS (Content Management Systems)

Webclient


1. a software the makes requests and receives web content
2. Requests a HTTP, receives HTTP, displays it in a user friendly UI

HTML


1. <head> saves metadata, title, stylesheets, scripts and meta tags
2. <!DOCTYPE html> tells the browser, which version of HTML is used
3. top level elements in <html> are <head> and <body>; the body contains the actual conent of the page
4. <strong> and <b> make the a text bold
  <em> sets your text italic
